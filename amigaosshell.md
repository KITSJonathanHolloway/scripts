Amiga Shell Commands
====================

See - http://wiki.amigaos.net/wiki/AmigaOS_Manual:_AmigaDOS_Command_Examples

Modify S:ShellCommands for aliases

Close the prompt

    endcli
  
Edit a file

    ed <filename>
  
Copy a file

    copy <file> TO <location>
    
Change up a directory:

  /
